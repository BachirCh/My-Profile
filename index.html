Process

  • Set the information architecture according to users mental model by doing a card sorting exercise with ±10 users, already prepared the template but we should check it together to make sure all sections are included in it.
  • Elaborate design system + Lo-fi/Hi-fi prototypes (readapting the design system to fit to the mobile version, switch to 8pt grid, ...)
  • Conduct usability testing to test prototypes (±5users for each prototype), gather insights, iterate to improve and then test again (We should prepare a list of participants in advance, and create a motto for them to participate)
  • Iterate, improve and test again until we are convinced
  • Present to stakeholders and launch development
